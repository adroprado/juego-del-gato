# Juego del Gato

### Descripción

Este código implementa una aplicación de un juego completo de Tres en Raya (Gato) que maneja el estado del tablero, el marcador, la detección de victoria/empate y el reinicio automático de la partida.

### Herramientas

- _HTML_
- _CSS_
  - ❌ _Responsive_
- _JavaScript_
  - _DOM_
